COMMANDES UTILES
================================================================================

Accéder au site:
  curl http://localhost:5173/www.melaniezufferey.ch/
  # Ou ouvrir dans un navigateur

Vérifier Apache:
  sudo /etc/init.d/apache2 status
  sudo /etc/init.d/apache2 restart
  sudo /etc/init.d/apache2 stop

Voir les logs:
  sudo tail -f /var/log/apache2/static-site-error.log
  sudo tail -f /var/log/apache2/static-site-access.log

Archiver le site:
  tar -czf wordpress-statique-backup.tar.gz www/
  zip -r wordpress-statique-backup.zip www/

Copier sur USB:
  cp -r www/ /mnt/usb/
  # (À adapter selon votre clé USB)

Réaspirer le site:
  rm -rf www/
  python3 scrape.py

================================================================================
SUPPORT ET TROUBLESHOOTING
================================================================================

Problème: "Le site ne s'affiche pas"
  → Vérifiez: sudo /etc/init.d/apache2 status
  → Redémarrez: sudo /etc/init.d/apache2 restart
  → Consultez: /var/log/apache2/static-site-error.log

Problème: "Les styles CSS ne s'appliquent pas"
  → Les fichiers CSS sont dans www/www.melaniezufferey.ch/wp-content/themes/
  → Vérifiez les chemins relatifs dans le HTML

Problème: "Les images ne s'affichent pas"
  → Les images sont dans www/www.melaniezufferey.ch/wp-content/uploads/
  → Vérifiez les chemins d'images

Pour plus d'aide:
  → Consultez manuel.md (section Dépannage)
